## About
DueMate is a task management app for iOS that helps users organize and track their tasks. Users can create tasks with due dates, categories, and priorities, and the app visually highlights overdue and upcoming tasks.

## Requirements
- Xcode 15+
- iOS 17.0+
- SwiftUI

## How to Run
1. Open `DueMate.xcodeproj` in Xcode
2. Select an iPhone simulator
3. Press Cmd+R to build and run
